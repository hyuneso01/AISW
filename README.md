# 프로젝트 제목: 스트레스 자가진단 웹앱

## 📌 개요
심리학과 지식을 바탕으로 사용자의 스트레스 정도를 체크하고 관리할 수 있는 웹앱입니다.

## ⚙️ 주요 기능
- 질문지 기반 스트레스 점수 계산
- 점수에 따른 결과 메시지 출력
- localStorage를 활용한 결과 저장

## 🛠 기술 스택
- HTML, CSS, JavaScript (Vanilla)
- localStorage

## 🚀 실행 방법
1. index.html 열기
2. 질문에 답변하면 결과가 바로 표시됩니다.
