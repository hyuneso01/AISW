# AISW
AISW수업 과제 제출용입니다.


프로젝트 제목: 간편 재무비율 분석 서비스


📌 개요<br/>
기업경영학과 전공지식을 바탕으로 기업의 간편 재무비율을 분석할 수 있는 웹입니다.<br/>
재무제표 분석에 익숙하지 않은 사용자에게 간단한 재무데이터를 입력하여 장단기 투자여부를 분석해줍니다.<br/>
간편 재무비율 분석 서비스를 HTML, CSS, JavaScript로 구현하였습니다. <br/>
사용자는 회사별 재무 데이터를 입력하고, 재무비율을 확인할 수 있으며 투자판단 결과를 제공합니다.<br/>
시각적으로 원그래프를 통해 비율을 직관적으로 이해하도록 했습니다.<br/><br/>

⚙️ 주요 기능
- 각 회사의 재무비율(유동비율,부채비율) 자동 계산
- 유동비율과 부채비율을 한눈에 볼수 있도록 시각화(그래프)자료 제공
- 재무비율을 조건함수에 맞게 투자판단 및 세부내용(단기,장기투자) 결과 제공
- 각 회사의 재무데이터 수정 및 삭제기능<br/><br/>
  
💡 실행 방법
1. index.html 열기 
2. 데이터입력 테이블에 초기화 버튼을 누른후 회사명, 유동자산, 유동부채, 총부채, 자본을 넣고 저장 버튼을 누른다.
   ex) 회사명: 삼성전자, 유동자산: 150, 유동부채: 50, 총부채: 60, 자본: 130 -> [분석결과] 장단기 투자가 모두 가능한 투자적합으로 결정됨(유동비율: 300%, 부채비율: 46.2%)
4. 시각화자료 및 데이터목록 테이블에 투자판단, 단기투자, 장기투자의 결과를 확인한다
5. 원하는 기업의 행을 클릭해서 회사의 데이터를 한눈에 확인할 수 있다.
6. 수정버튼을 눌러 기존 회사 데이터를 변경할 수 있고, 불필요시 삭제버튼으로 데이터를 지운다.<br/><br/>

🛠 사용한 기술스택
- HTML, CSS, JavaScript (chart.js)
- localStorage
- 핀터레스트 이미지 배경 적용
- 데이터 목록에 마우스 호버 적용
- 모바일 반응형 적용<br/><br/>

주의-투자 신중 : 유동비율 > 200% AND 부채비율 > 100% <br/>

가능-투자 적합 : 유동비율 > 200% AND 50% < 부채비율 ≤ 100% <br/>
가능-투자 적합 : 유동비율 > 200% AND 부채비율 ≤ 50% <br/>
