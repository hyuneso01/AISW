# 간편 재무비율 분석 서비스 PRD

## 개요
간편 재무비율 분석 서비스를 HTML, CSS, JavaScript로 백엔드 없이 로컬스토리지를 사용하여 구현한다. 사용자는 회사별 재무 데이터를 입력하고, 각종 재무비율(유동비율, 부채비율)을 확인할 수 있으며, 시각적으로 원그래프를 통해 비율을 직관적으로 이해할 수 있다.

## 주요 기능

### 1. 데이터 입력
- **회사명 입력 박스**
- **유동자산 금액 입력 박스**
- **유동부채 금액 입력 박스**
- **총부채 금액 입력 박스**
- **자본금액 입력 박스**
- **유동비율 계산 박스** (자동 계산)
- **부채비율 계산 박스** (자동 계산)

### 2. 데이터 관리
- **수정 기능**: 입력된 데이터를 선택 후 수정 가능
- **삭제 기능**: 입력된 데이터를 선택 후 삭제 가능
- **로컬스토리지 저장**: 새로고침 후에도 데이터 유지

### 3. 재무비율 계산
- **유동비율**: `(유동자산 ÷ 유동부채) × 100` → 소수점 한 자리 표시
- **부채비율**: `(총부채 ÷ 자본) × 100` → 소수점 한 자리 표시

### 4. 시각화
- **유동비율 원그래프**: 유동자산과 유동부채를 비교
- **부채비율 원그래프**: 총부채와 자본을 비교

### 5. UI/UX
- 입력 박스와 계산 결과, 원그래프를 한 화면에 표시
- 수정/삭제 버튼을 각 데이터 항목 옆에 배치
- 데이터 입력, 수정 시 실시간 계산 및 그래프 업데이트

## 기술 스택
- **Frontend**: HTML, CSS, JavaScript
- **저장소**: 브라우저 로컬스토리지
- **그래프 라이브러리**: Chart.js (혹은 Canvas API 사용 가능)

## 요구사항 요약
| 기능 | 상세 설명 |
|------|-----------|
| 데이터 입력 | 회사명, 유동자산, 유동부채, 총부채, 자본 입력 가능 |
| 계산 기능 | 유동비율, 부채비율 자동 계산 |
